# Introduction to WebGL
This is an introduction to WebGL2 using [Umbra](https://github.com/Lakuna/Umbra).
The contents of this series of tutorials is mostly reorganized from the excellent content at [WebGL2Fundamentals](https://webgl2fundamentals.org).

All concepts will first be demonstrated directly using the WebGL API before being demonstrated using Umbra.
In other words, this series will serve as a proper tutorial to learn WebGL even if you don't plan on using Umbra.

## Table of Contents

2. [Fundamentals](/webgl/fundamentals)
3. [Structure](/webgl/structure)
