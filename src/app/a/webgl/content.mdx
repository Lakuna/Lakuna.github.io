# WebGL2 Tutorial

This is a tutorial series for WebGL2. It is based on [WebGL2 Fundamentals](https://webgl2fundamentals.org/) by Gregg Tavares, [Learn OpenGL](https://learnopengl.com/), and [Maxime Euzière's WebGL guide](https://xem.github.io/articles/webgl-guide.html).

Many examples use [μGL](https://ugl.lakuna.pw/) and/or [μMath](https://umath.lakuna.pw/) after their concept has been demonstrated without a library in order to reduce their complexity.

See the glossary [here](/a/webgl/glossary).
