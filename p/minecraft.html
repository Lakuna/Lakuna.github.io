How to Join


Ensure that you have a 64-bit copy of Java 8 installed. 32-bit Java allows a maximum of 1 gigabyte of allocated memory, which makes modded Minecraft nearly impossible. Versions of Java newer than 8 don't work with versions of Forge prior to Minecraft 1.13. The 64-bit installer for Java 8 can be found here (for Windows users, click Windows Offline (64-bit)).

Download the recommended version of MCUpdater here. MCUpdater is the client you will use to run Minecraft with my modpack.

Upon launching MCUpdater, it will ask you to log in to Minecraft. Note that, although it asks for your username, you must log in using your email address.

In the top-left corner of MCUpdater, click the green circle with a plus inside of it to add a pack. Use the following URL as the ServerPack XML:

https://lakuna.pw/s/mc/pack.xml

Click Update in the bottom-right corner of MCUpdater to download the mods, then you can connect to the server using the Launch Minecraft button.

Manual Connect


If you cannot get MCUpdater to work, you can download all of the mods independently and join the server through a normal Forge profile on the Minecraft launcher. This will be tedious and time-consuming.

First, install Forge normally on their website. The server uses Forge version 14.23.5.2844.

Navigate to the ServerPack XML. Each mod that the server uses has an entry in the XML file, including a URL to the mod's main jar file. Download each mod from the given link.

Install the downloaded mods normally. You can then connect to the server at IP mc.lakuna.pw.
