Application Data

Application data from my programs is always stored in ~/.t3, where ~ is the home directory.

C#

string path = Environment.GetEnvironmentVariable("USERPROFILE") + "/.t3";

Java

String path = System.getProperty("user.home") + "/.t3";

Node.js

const path = require('os').homedir() + '/.t3';

Python

from pathlib import Path
path = str(Path.home()) + '/.t3'

QB64

DECLARE DYNAMIC LIBRARY "shell32"
    FUNCTION SHGetFolderPathA (BYVAL a, BYVAL b&, BYVAL c%&, BYVAL d, BYVAL e%&)
END DECLARE
path$ = STRING$(260, 0)
a = SHGetFolderPathA(0, &H28, 0, 0, _OFFSET(path$))
path$ = LEFT$(path$, INSTR(path$, CHR$(0)) - 1) + "/.t3"
PRINT path$
