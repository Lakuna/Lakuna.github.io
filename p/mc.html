Sortaflat

See the original post here.

Sortaflat is a custom world type which generates normal vanilla biomes in a superflat world. There are two versions available: one is "pure" sortaflat, which has no variation in block height like a true superflat world, and the other is "dirty" sortaflat, which allows about one block of variation so that rivers and oceans contain water.
